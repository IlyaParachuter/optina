# optina bible browser for mobile
Этот проект использует данные с сайта 'http://bible.optina.ru'

datastore-manager написан на python 2.7 (windows).
browser написан для платформы midp 2.0 . Использует старый движок, при просмотре больших страниц придется дождаться их загрузки.

Проект находится на стадии разработки.
